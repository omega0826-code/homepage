타이타닉 생존자 예측 (분류)

데이터 전처리, 피처 엔지니어링, 여러 분류 모델 비교 및 성능 튜닝을 통해 생존 가능성을 예측했습니다. 모델 설명을 위한 SHAP 분석을 통해 피처 중요도를 해석했습니다.

주요 작업

  • 결측치 처리 및 범주형 피처 엔지니어링
  • 여러 분류 모델(랜덤포레스트, XGBoost, 로지스틱 회귀) 비교
  • 하이퍼파라미터 튜닝 및 교차검증
  • SHAP 기반 모델 설명과 피처 영향도 시각화

코드 및 노트북

실습 노트북: GitHub 리포지토리

결과 요약
